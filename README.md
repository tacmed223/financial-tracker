# 💰 Financial Tracker Pro

A simple and modern personal finance tracker that lets you log income and expenses, automatically calculates your balance, and saves everything in your browser.

---

## 🚀 Features

- Add income and expense transactions
- Automatic balance calculation
- Live transaction table
- Search/filter transactions
- Delete transactions
- Saves data using **localStorage** (persists after refresh)

---

## 🖥️ How It Works

1. Enter a description (e.g. Salary, Food, Rent)
2. Enter an amount
3. Choose date
4. Select type (Income or Expense)
5. Click **Add Transaction**

Your data will automatically:
- Update totals
- Show in the table
- Save in your browser

---

## 📦 Tech Used

- HTML
- CSS (embedded styling)
- JavaScript (Vanilla JS)
- localStorage (for saving data)
